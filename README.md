## 🛠️ Common Installation & Local Setup Guide

The procedure for running all projects on the local machine is almost the same.:

1.  **Clone the Repo:**
    ```bash
    git clone <repository-url>
    ```
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Environment Setup:**
    
One in the root directory `.env` or `.env.local` Create the file and your **Firebase & Database** Add the credentials.
4.  **Run Development Server:**
    *   **React/Next.js:** `npm run dev` or `npm start`
    *   **Vue:** `npm run dev`
    *   **Backend:** `node index.js` or `nodemon index.js`
